# Grazioso Salvare Animal Shelter CRUD System

**Overview**
This project was developed in CS-340: Client/Server Development.  
The application provides a Python interface for interacting with a MongoDB database containing animal shelter records from the Austin Animal Center dataset.

The system was designed to support Grazioso Salvare, a rescue organization that trains animals for search and rescue operations.

**Features**

- Python client application
- MongoDB CRUD operations
- Query filtering for rescue candidates
- Jupyter notebook testing environment
- Data visualization dashboard milestone

**Technologies Used**
- Python
- PyMongo
- MongoDB
- Jupyter Notebook
